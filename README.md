# HeartSavers Africa

**Empowering Communities to Save Lives**

A multi-page static website for HeartSavers Africa — a nonprofit providing free, in-person CPR training to communities and schools across the U.S. and Nigeria.

## Pages
- **Who We Are** — Mission, founders, NREMT certification, partners
- **What We Do** — Training details, feature cards, no-certification notice
- **Events** — Upcoming sessions, registration, pre/post training assessments
- **Blog** — CPR tips and first aid articles

## Tech Stack
- HTML5, CSS3, JavaScript (no frameworks)
- Google Fonts (Inter)
- Fully responsive with mobile hamburger menu

## Getting Started
Open `index.html` in your browser.
